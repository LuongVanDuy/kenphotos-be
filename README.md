## Trạng thái bài viết (`Post.status`)

| Giá trị số | Tên trạng thái | Diễn giải                  |
| ---------- | -------------- | -------------------------- |
| 0          | DRAFT          | Bản nháp                   |
| 1          | PUBLISHED      | Đã xuất bản                |
| 2          | PRIVATE        | Riêng tư                   |
| 3          | PASSWORD       | Bảo vệ bằng mật khẩu       |
| 4          | TRASH          | Đã xoá (gửi vào thùng rác) |
